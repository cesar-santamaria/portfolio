## How It's Made:

**Tech used:** React NextJS

## Dependencies
React
Next
Framer-motion
chakra-ui

## Getting Started
Install dependencies using the npm install command within each folder directory.

- Start the web server using the npm run dev. The site will be served on http://localhost:3000/.
